Graph class

Constructor

ArgumentDefault valueNotes
nameThe graph's name.
layoutName"default"The name of the layout to use for the graph.

Attributes

AttributeClassDefault valueNotes
typeString"Graph"The class name.
nameString""The Graph object's name.
nodesArray of NodesAn array of nodes in the graph.
edgesArray of EdgesAn array of edges connecting nodes in the graph.
svgSVG GroupThe SVG group object representing this graph on the HTML page.
borderArray of 3 Integers[100, 100, 0]Three-element array; nodes are not placed within this distance of the sides of the window.
layoutLayoutA Layout object.
allowSelfEdgesBooleanTrueWhether edges from a node to itself are allowed to be created.
alwaysUseBezierBooleanTrueWhether new edges are straight lines or Bezier curves.
degreeMatrixArray of Integers[]A square array whose diagonal values contain the degree of each node in the graph (other entries are undefined); empty until updated.
adjacencyMatrixArray of Booleans[]A square array whose [i][j]th entry is true if the ith and jth nodes are connected by at least one edge; empty until updated.
distanceMatrixArray of Integers[]A square array whose [i][j]th entry is the number of steps from the ith node to the jth node; empty until updated.
connectivityMatrixArray of Booleans[]A square array whose [i][j]th entry is true if the jth node can be reached from the ith node; empty until updated.
connectedComponentsArray of Arrays of Nodes[]An array whose entries are arrays of connected components of the graph; empty until updated.
